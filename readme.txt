== Description ==

A simple plugin that add widget with temperature and humidity from www.tmep.cz.

For setup widget see screenshots.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/widget-for-tmepcz' directory, or install the plugin through the WordPress plugins screen directly
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add the 'TMEP.cz widget' to your widget area
1. Set the widget
